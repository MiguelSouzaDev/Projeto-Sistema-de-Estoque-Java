# Sistema de Estoque em Java
Projeto simples de sistema de estoque feito em Java, com foco em lógica de programação e Programação Orientada a Objetos.

## Funcionalidades
- Listar produtos
- Buscar produto por ID
- Remover produto do estoque
- Menu interativo via console

## Tecnologias
- Java
- Scanner
- List / ArrayList

## Estrutura
- Produto.java: representa o produto
- Estoque.java: gerencia os produtos
- Main.java: interface com o usuário

## Como executar
Compile e execute com:

javac *.java
java Main
